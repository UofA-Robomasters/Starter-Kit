### Mac OS
in terminal

```commandline
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

git clone https://github.com/tzutalin/labelImg.git
cd labelImg

brew install qt
brew install libxml2
make qt5py3
python labelImg.py
```
